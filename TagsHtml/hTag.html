Hierarquia da tag H

O uso das tags h's de 1 a 6 é uma das mais utilizadas na criação de websites. Atualmente é possível utilizar do valor 1 ao 6 da tag, sendo também recomendável usar em ordem, como exemplo ao usar uma tag h1 o recomendável é utilizar em seguida a tag h2 e assim por diante, portanto deve-se saber para qual motivor usar cada valor, quanto menor o valor de maior destaque será o texto dentro dela, abaixo está demonstrado a utilização das tags h1 á h6 em ordem para se ver a diferença entre elas.

Como visto acima quanto maior o valor atribuido a tag menor será o destaque, por isso é necessário saber qual tag usar na hora certa.

Na criação desta documentação foram utilizadas basicamente as tags h's, caso queira visualizar elas apenas pressione o botão f12 do teclado, ou cicar com o botão direito na tela e clicar em inspecionar elemento e clicar na aba Elements, assim sendo possível ver as tags utilizadas para criar este site.
