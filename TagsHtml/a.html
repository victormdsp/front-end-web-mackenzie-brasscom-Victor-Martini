Tag ‹a› e seu funcionamento

O intuito da Tag ‹a› é criar um método de acessar diferentes links para outras páginas principalmente.

Para o funcionamento da tag ‹a› é preciso colocar como atributo um href, que é traduzido como HyperText Reference, ou seja, ela referencia a algum outro HTML.

Seu uso é feito da seguinte forma:

‹a href = "link para a página qual deseja acessar"›

Link exemplo que ao clicar o usuário será redirecionado para o youtube.
