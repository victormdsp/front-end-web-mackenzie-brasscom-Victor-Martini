Tag ‹ul›

O uso da tag ‹ul› é para a criação de listas não-ordenadas, como mostrado no exemplo a baixo.

  • Isto é uma tag ‹li›
  • Isto é outra tag ‹li›

Como mostrado no exemplo acima a tag ‹li› é um elemento de uma lista não-ordenada que está dentro de uma tag ‹ol›, como da para ver acima a lista não possui uma numeração ou algo que indique uma ordem na lista, assim sendo considerada uma lista não-ordenada.
