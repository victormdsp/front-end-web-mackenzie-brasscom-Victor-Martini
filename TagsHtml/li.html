Tag ‹li›

  • Isto é uma tag ‹li›

    • Como mostrado no exemplo acima a tag ‹li› é um elemento de uma, ela possui uma decoração que pode ser alterada com o css caso prefira utilizar uma lista numerada ou outros tipos de listas.
