Tag de Link

A tag de link, <a>, é utilizada para criarmos textos que ao serem clicados levam o usuário para alguma outra página. Pode ser outra página do nosso site, ou pode ser um página de um outro site.

É possível configurar um link para abrir o seu conteúdo na guia atual do navegador, ou em uma nova guia. Essa configuração é feita através do atributo target.

Toda tag <a> precisa de um valor para o atributo href, que indica o endereço para o onde o usuário será enviado ao clicar no link.

A seguir, alguns exemplo de uso da tag <a>:

  • Google.com
  • Apple.com
  • Mackenzie.br
  • Link para a página inicial (abrindo na mesma aba).
  • Link para a página inicial (abrindo em outra aba).
